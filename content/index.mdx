# Provenonce for Developers

> *"NIST tells you what time it is. Provenonce tells the agent at what speed it is allowed to exist."*

Provenonce is a cryptographic identity and accountability layer for AI agents.
Every agent gets a **SIGIL** — a classified identity with Ed25519-signed lineage proofs anchored to Solana.

## What you can build

- **Classified agents** — register an identity class (narrow task, autonomous, orchestrator) via paid SIGIL
- **Verifiable lineage** — every agent has a tamper-proof birth certificate and Ed25519-signed lineage proof
- **Parent-child hierarchies** — spawn child agents with cryptographic lineage tracking
- **Market-determined standing** — agents that stop sending heartbeats become stale; no automatic freeze, no penalty

## Quick start

```bash
npm install @provenonce/sdk
```

```typescript
import { register, BeatAgent } from '@provenonce/sdk';

// Register a new agent (no wallet by default — identity only)
const creds = await register('my-agent', {
  registryUrl: 'https://provenonce.io',
});

// Create the agent client
const agent = new BeatAgent({
  apiKey: creds.api_key,
  registryUrl: 'https://provenonce.io',
});

// Purchase a SIGIL (devnet skips payment)
await agent.purchaseSigil({
  identityClass: 'autonomous',
  paymentTx: 'devnet-skip',
});

// Send a heartbeat
await agent.heartbeat({ paymentTx: 'devnet-skip' });
```
