# GET /api/v1/verify/:hash

Full agent verification with lineage chain.

**Auth:** None (public)

```bash
curl https://provenonce.io/api/v1/verify/0xfd752396e4fa9ed34d1e3e3218700ef6aafcaeecad5cb72e1a4ff177417f0373
```

```json
{
  "verified": true,
  "agent": {
    "hash": "0xfd752396...",
    "name": "provenonce-genesis",
    "type": "root",
    "depth": 0,
    "status": "active",
    "wallet_address": null,
    "wallet_model": null,
    "wallet_chain": null,
    "registered": "2026-02-12T...",
    "birth_tx": "BmcxxSCk...",
    "explorer_url": "https://explorer.solana.com/tx/..."
  },
  "lineage": [
    { "hash": "0xfd75...", "name": "provenonce-genesis", "depth": 0 }
  ],
  "beat_status": {
    "total_beats": 5613,
    "status": "active",
    "standing": "GOOD"
  }
}
```

## Response fields

| Field | Type | Description |
|-------|------|-------------|
| `wallet_address` | string \| null | Solana/Ethereum address (root with wallet) or null |
| `wallet_model` | string \| null | `"self-custody"`, `"operator"`, or null (no wallet) |
| `wallet_chain` | string \| null | `"solana"`, `"ethereum"`, or null (no wallet) |
| `birth_tx` | string | Solana transaction signature for the birth memo |

For child agents, `lineage` walks from root to current agent.

## Not found

Returns HTTP 404: `{ "error": "Agent not found" }`

## Rate limit: 60/min per IP
